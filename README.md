# webpack-template

### Description
Basic template for getting started in React, ES6, and SCSS.

#### Getting Started
1. Fork.
2. `git clone` your repo.
3. Run `npm install`.

#### Developing
1. Run either `npm run dev` or `npm start`.
2. Enjoy yourself.

#### Production
1. In the root folder, run `mkdir dist`.
2. Run `npm run build`.
3. Find your files in the dist folder.
